# 🚀 News Tech 2026: Project Walkthrough

I have successfully developed a professional, front-end News Technology website tailored for 2026 standards. The project uses high-performance HTML5, CSS3, and modern JavaScript to deliver a premium user experience.

## ✨ Key Deliverables

### 1. 📂 Content Architecture (10 Pages)
Total of 10 comprehensive pages each containing unique news content (16+ items each):
- **Home**: Main portal with hero section.
- **News**: Breaking Tech headlines.
- **About**: Company insights and journey.
- **Service**: Enterprise tech solutions.
- **Contact**: Support and inquiry channels.
- **Trends**: Emerging market trends.
- **Technology**: Core hardware and software innovations.
- **Innovations**: Breakthrough inventions.
- **Future**: Life in 2030 and beyond.
- **Security**: Cyber-protection and data privacy.

### 2. 🔐 Authentication & Dashboard
- **Sign-In Page**: Features a modern card design with login methods for **Admin** and **Customer**.
- **Role-Based Dashboard**: Automatically filters content based on the login method. 
  - **Admin View**: Features an interactive analytics board, server status, and logs.
  - **Customer View**: Shows personal article saves and subscription status.

### 3. 🛡️ Safety Redirection (404 Error Page)
To simulate incomplete or under-construction features, all interactive buttons (e.g., "Read More") redirect to a branded **404 Page**. This ensures a consistent user experience during the preview phase.

### 4. 🎨 Design System
- **Advanced Palette**: A professional mix of Deep Indigo (#1a2a6c) and Vibrant Crimson/Yellow gradients.
- **Modern Typography**: Inter font family for maximum readability.
- **Glassmorphism**: Navbar and cards utilize backdrop-blur for a 2026 aesthetic.
- **Responsive**: Fully optimized for mobile (min-width 320px).

### 5. 🌍 Universal Footprint
- **Logo Consistency**: The specified `Logo1.webp.PNG` is integrated via a global common script.
- **Salem Stackly Map**: A Google Maps integration pointing to the Salem Stackly address is included on every page.

---

## 🛠️ Instructions for Use
1. **Launch**: Open `index.html` in your browser.
2. **Explore**: Navigate using the menu links (Home, News, About, etc.).
3. **Login**: Go to the **Sign In** page, select "Administrator" or "Customer", and click "Sign In" to see the filtered dashboard.
4. **Button Testing**: Click any "Read More" button to see the automatic redirect to the 404 page.

> [!TIP]
> To use your own logo, simply place a file named **Logo1.webp.PNG** in the root directory, and it will automatically appear on all pages.
