# Stackly 2026 - Technology News Portal

Welcome to the **Stackly 2026** technology portal. This is an industry-level front-end application built with only HTML, CSS, and JavaScript.

## Project Features
- **10 Distinct Content Pages**: Including News, Trends, Future Tech, and Security.
- **Dynamic Content Injection**: Each page displays 16 unique articles with 2026-level high-quality WebP imagery.
- **Universal Header & Footer**: Modern navbar with hover animations and localized Google Maps.
- **Role-Based Authentication**: Custom sign-in logic that redirects to an Admin or Customer dashboard.
- **Responsive Design**: Flawless experience from 320px (mobile) to ultra-wide displays.
- **Advanced Aesthetics**: Premium indigo/accent color theme (non-black) with glassmorphism and smooth transitions.
- **Safety Redirection**: All interaction buttons (except login) redirect to a custom 404 page for preview purposes.

## Directory Structure
- `index.html`: Entry point (Home).
- `signin.html` / `dashboard.html`: Authentication flow.
- `404.html`: Universal error redirection page.
- `news.html`, `about.html`, `service.html`, etc. (10 Category Pages).
- `style.css`: The design system.
- `data.js`: Central content store (160+ unique items).
- `common.js`: Global logic, header/footer injection, and button handling.

## How to use
Simply open `index.html` in any modern browser.
